# Research Project's Structure:				Variable Name (Country Scope): 						Notes:
# Project ID								ag_research_project_[seq]							0 means no project.
# Project Status 							ag_research_project_[seq]_status					0 ~ 3, shouldn't be set here.
# Project Difficulty						ag_research_project_[seq]_difficulty				Very Easy (<2) ~ Hard (>=5)
# Project Requirements						ag_research_project_[seq]_requirements				1: 1 Science ship, 2: 1 Construction ship
# Project Area								ag_research_project_[seq]_area						1 ~ 3: Phy ~ Eng, 4: Phy & Soc, 5: Phy & Eng, 6: Soc & Eng, 7: All
# Project Required Progress					ag_research_project_[seq]_required_progress			-
# Project Total Progress					ag_research_project_[seq]_total_progress			-
# Project Research Counter					ag_research_project_[seq]_research_counter			0 ~ 6, use to avoid crash, shouldn't be initialized or changed.
####################################################################################################################################################################

# (10) Create context switch effect in scripted effect.

# (12) Create research project cmpleted effect.
